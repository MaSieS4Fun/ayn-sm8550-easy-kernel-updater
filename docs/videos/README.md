# Demo videos

Add your two ~10 second comparison clips here before publishing:

| File | Description |
|------|-------------|
| `demo-tuned-kernel.mp4` | Gameplay with kernel built by **this project** (good FPS, smooth) |
| `demo-armbian-default.mp4` | Same game/settings with **default Armbian kernel config** (stutters, ~40–50% loss) |

Recommended: same title, same scene, same settings — only the kernel differs.

After upload, the main [README](../README.md) embeds them automatically if these filenames are used.

**GitHub tip:** You can also drag MP4 files into the README editor on github.com; GitHub will host them under `user-attachments/assets/`.
